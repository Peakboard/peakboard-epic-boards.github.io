# 可使用的数据源

该模板使用简单的peakboard变量表作为数据源。它可直接访问您的系统。可使用的系统和数据格式包括：JSON，XML，CSV，REST-API，ODBC，MS SQL Server, MS Access，Azure，Oracle，MySQL，PostgreSQL，MongoDB和[许多其他数据源](https://peakboard.com/en/data-connections/)。
